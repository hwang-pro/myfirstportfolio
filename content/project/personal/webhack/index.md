## 🎯 프로젝트 소개

## 🔧 사용 기술
- **취약점 분석**: OWASP Top 10, CVE 분석

- **공격 기법**: SQL Injection, XSS, CSRF, SSRF
- **방어 기법**: WAF, Input Validation, Output Encoding
