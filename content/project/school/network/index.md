## 🎯 학습 목표

네트워크의 기본 개념과 실제 네트워킹 기술을 학습합니다.

## 📚 학습 내용
- **네트워크 기본 개념**: OSI 7계층, TCP/IP 모델
- **프로토콜**: HTTP, HTTPS, FTP, SSH
- **라우팅과 스위칭**: 네트워크 장비 이해
- **보안**: 방화벽, VPN, 암호화

## 🔧 실습 프로젝트
- 네트워크 토폴로지 설계
- 패킷 분석 (Wireshark)
- 서버 간 통신 구현
- 네트워크 보안 설정

## 📈 학습 성과
- 네트워크 아키텍처 이해
- 프로토콜 분석 능력
- 네트워크 보안 인식 향상
